# MosquesTN Mobile

Expo app scaffolded to match `.salihoon` theme.

## Quick Start

```bash
npm install
npm run start
```

## Theme
- Colors, spacing, and typography are in `src/theme.js`.
- Use `ThemedView` for consistent backgrounds.
